HTML and CSS Resource Guide

HTML Tags

  • The HTML <b> tag defines a bold text.
  • The HTML <i> tag defines an italic text.
  • The HTML <u> tag defines an underlined text.
  • The HTML <sub> tag defines a subscript text.
  • The HTML <sup> tag defines a supercript text.
  • The HTML <mark> tag defines a highlighted text.
  • The HTML <p> tag defines a paragraph
  • The HTML <small> tag defines a smaller text.
  • The HTML <strong> tag defines an important text.
  • The HTML <ul> tag defines an unordered list.
  • The HTML <ol> tag defines an ordered list.
  • The HTML <li> tag defines a list item.
  • The HTML <br> tag produces a single line break.
  • The HTML <hr> tag produces horizontal line to break text.
  • The HTML <a> tag defines a hyperlink.
  • The HTML <!--...--> tag define a comment.
  • The HTML <h1-h6> tags define HTML headings.
  • The HTML <blockquote> tag indents a block of text.
  • The HTML <em> tag defines emphasized texts.
  • The HTML <dl> tag defines a description list.

CSS Properties

  • The CSS background-color property specifies the background color of an element.
  • the CSS color property specifies the color of text content for an element.
  • the CSS font-family property specifies the font of a text.
  • the CSS font-size property specifies the font size of a text.
  • the CSS font-style property specifies the font style of a text.
  • the CSS font-weight property specifies the thickness of a text.
  • the CSS text-align property specifies horizontal alignment of a text.
  • the CSS text-decoration property specifies the decoration added to a text.
  • the CSS padding property specifies the space between its elements and border.
  • the CSS line-height property specifies the height of a line box.
  • the CSS border property specifies an element's border.
  • the CSS border-color property specifies the color of an element's border.
  • the CSS text-indent property specifies the identation of the first line in a block of text.
  • the CSS text-shadow property specifies the shadow added to a text.
  • the CSS letter-spacing property specifies the white space between text characters.
  • The CSS word-spacing property specifies the white space between words.
