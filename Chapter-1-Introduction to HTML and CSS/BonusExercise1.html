HTML and CSS Resource Guide

CSS Properties

CSS Property Description Example
background-color property Specifies the background color of an element. background-color: lightpink;
color property Specifies the color of text content for an element. color: orangered;
font-family property Specifies the font of a text. font-family: 'Trebuchet MS';
font-size property Specifies the font size of a text. font-size: 20px;
font-style property Specifies the font style of a text. font-style: oblique;
font-weight Specifies the thickness of a text. font-weight: 900;
text-align property Specifies the horizontal alignment of a text. text-align: center;
text-decoration property Specifies the decoration added to a text. text-decoration: line-through;
padding property Specifies the space between its elements and border. padding: 10px;
line-height property Specifies the height of a line box. line-height: 50px;
border property Specifies an element's border. border: 1px solid purple;
border-color property Specifies the color of an element's border. border-color: solid purple;
text-indent property Specifies the indentation of the first line in a block of text. text-indent: 5px;
text-shadow property Specifies the shadow added to a text. text-shadow: 2px 2px #E5D1FA;
letter-spacing property Specifies the white space between text characters. letter-spacing: 2px;
word-spacing property Specifies the white space between words. word-spacing: 2px;
float property Specifies how an element is positioned relative to other elements. float:right;
font-variant property Specifies how an element is positioned relative to other elements. font-variant: small-caps;
