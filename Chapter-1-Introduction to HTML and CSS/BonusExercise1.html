HTML and CSS Resource Guide

CSS Properties

CSS Property Description
background-color property Specifies the background color of an element.
color property Specifies the color of text content for an element.
font-family property Specifies the font of a text.
font-size property Specifies the font size of a text.
font-style property Specifies the font style of a text.
font-weight Specifies the thickness of a text.
text-align property Specifies the horizontal alignment of a text.
text-decoration property Specifies the decoration added to a text.
padding property Specifies the space between its elements and border.
line-height property Specifies the height of a line box.
border property Specifies an element's border.
border-color property Specifies the color of an element's border.
text-indent property Specifies the indentation of the first line in a block of text.
text-shadow property Specifies the shadow added to a text.
letter-spacing property Specifies the white space between text characters.
word-spacing property Specifies the white space between words.
float property Specifies how an element is positioned relative to other elements.

HTML Tags

HTML Tag Description
<b> tag Defines a bold text.
<i> tag Defines an italic text.
<u> tag Defines an underlined text.
<sub> tag Defines a subscript text.
<sup> tag Defines a superscript text.
<mark> tag Defines a highlighted text.
<p> tag Defines a paragraph
<small> tag Defines a smaller text.
<strong> tag Defines an important text.
<ul> tag 1. Defines an unordered list.
<ol> tag ● Defines an ordered list.
<li> tag Defines a list item.
<br> tag
Produces a single line break.
<hr> tag Produces a horizontal line to break text.
<a> tag Defines a hyperlink.
<!--...--> tag Defines a comment.
<h1-h6> tags Defines HTML headings.
<blockquote> tag
Indents a block of text.
<em> tag Defines emphasized text.
<dl> tag Defines a description list.
